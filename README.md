# AGROSOFTWARE S.A.

### Conectando el campo con la tecnología

AGROSOFTWARE es un sistema de gestión para cooperativas agrícolas andaluzas, especializado en la digitalización de la recogida de aceitunas.

## Características principales
- Módulo móvil para registrar la entrada de aceitunas por parcela.
- Panel web para gestionar cooperativistas y lotes.
- Informes de producción y trazabilidad.
- Módulos premium de análisis y exportación (versión profesional).

## Modelo de negocio
Modelo mixto:
- El núcleo del sistema es software libre bajo licencia MIT.
- Los módulos de análisis avanzado y exportación son de pago.
- Ofrecemos servicios de instalación, soporte y personalización para cada cooperativa.
